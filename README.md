# Seth Steels Enterprises

A premium, modern B2B website for Seth Steels Enterprises, a leading industrial steel supplier.

## Features
- **Modern Design**: Built with Tailwind CSS and Inter typography.
- **Responsive**: Fully optimized for mobile, tablet, and desktop views.
- **Interactive**: Scroll-triggered animations and smooth navigation.
- **Conversion Focused**: Strategically placed CTAs and trust indicators.

## Tech Stack
- HTML5 / CSS3
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Font Awesome Icons

## Local Development
Simply open `index.html` in any modern web browser.

## Deployment
This site is designed for easy deployment to **Cloudflare Pages**.
1. Push this repository to GitHub.
2. Connect the repository to Cloudflare Pages.
3. Use `./` as the build output directory and no build command.
